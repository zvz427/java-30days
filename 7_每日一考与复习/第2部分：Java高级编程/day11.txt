5. 如何调用方法show()  
类声明如下：
package com.atguigu.java;
class User{
	public void show(){
		System.out.println(“我是一个中国人！”);  
}  }

User user = (User)clazz.newInstance();
Method show = clazz.getDeclaredMethod("show");
show.setAccessiable(true);
show.invoke(user);
